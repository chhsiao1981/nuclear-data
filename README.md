核電翻譯機 (Nuclear-Data)
=========================

How to run?
-----------

Install nodejs / ruby
    $ brew install node ruby
    $ npm install bower -g
    $ gem install compass

Install local dependencies

    $ npm install
    $ bower install

Run the following command:
    
    $ npm run dev

It will start python server and a livereload server on 'static', which will compile gh-pages to 'static'
